# Typing-Speed-Test-Python
Typing-Speed-Test-Python is a desktop application built with Python and Tkinter to help users practice and improve their typing speed and accuracy. The app generates 300 random sentences, each containing 20-24 words, and challenges users to type them as quickly and accurately as possible.

Features
Random Sentence Generation: 300 unique sentences are created from a pool of common words for varied practice.
Real-Time Feedback: Instantly see your typing speed (WPM) and accuracy after each attempt.
Simple GUI: Clean and user-friendly interface using Tkinter.
Progress Tracking: Displays time taken and words per minute for each test.
Easy to Use: Just press Start, type the displayed sentence, and get your results.
How It Works
Click "Start" to begin the test.
A random sentence appears. Type it exactly in the input box.
Press Enter to submit your attempt.
The app checks your input, calculates your WPM and time, and displays feedback.
Try again to improve your speed and accuracy!
Requirements
Python 3.x
Tkinter (usually included with Python)
Usage
Clone the repository and run the script:

python import tkinter as tk.py

This project is ideal for anyone looking to improve their typing skills, challenge themselves, or simply have fun with a typing game.
